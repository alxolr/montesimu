# Documentation

This folder contains all Kiro-related documentation, requirements, and project specifications.

## Structure

- `requirements/` - Project requirements and feature specifications
- `steering/` - Kiro steering files for workflow guidance
- `specs/` - Detailed feature specs following spec-driven development
- `architecture/` - System architecture and design documents
- `guides/` - Development guides and best practices

## Usage

All Kiro specs, requirements documents, and steering files should be organized within this directory structure to keep the project documentation centralized and maintainable.
